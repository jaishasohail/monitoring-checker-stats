# Monitoring Checker Stats Scraper Usage Guide

This guide walks you through installing, configuring, and running the Monitoring Checker Stats Scraper locally.

## 1. Prerequisites

- Python 3.9 or newer
- Internet access if you want to monitor real HTTP endpoints
- `pip` available on your system

## 2. Installation

From the project root directory:

```bash
python -m venv .venv
source .venv/bin/activate  # On Windows use: .venv\Scripts\activate
pip install -r requirements.txt
